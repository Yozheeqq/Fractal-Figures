Данный проект разработан на WPF и предлагает построить фрактальные деревья
Есть возможность выбрать 5 видов фрактала, а также покрасить фрактал в градиент


Желательное значение для максимальной глубины рекурсии для каждого фрактала:
Фрактальное дерево: 15
Снежинка Коха:7
Ковер Серпинского: 6
Треугольник Серпинского: 12
Множество Кантора: 15

Также градиент плохо работает для Ковра Серпинского.

Для сохранения картинки нажать Файл в верхнем левом углу.
